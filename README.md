# Vending Machine.

A PHP coding exercise.

## Instruction how to run the environment

First run,

```bash
make init up
```

After that daily stopping
```bash
make down
```

and starting
```bash
make up
```

To run the unit tests
```bash
make test
```
# Usage

After running the docker,the API can be found at http://127.0.0.1/api/docs
